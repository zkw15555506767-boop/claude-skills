# 参考资料库

此目录存储你在学习中收集的参考资料，包括：
- 官方文档链接
- 教程和指南
- 有用的代码片段
- 学习笔记

## 分类组织

- `links.md` - 常用链接收藏
- `snippets/` - 代码片段
- `notes/` - 学习笔记

## 添加新资料

### 添加链接
在 `links.md` 中按类别添加链接：

```markdown
## 类别名称

- [链接标题](URL) - 简要说明
```

### 添加代码片段
在 `snippets/` 目录下创建新的片段文件。

### 添加笔记
在 `notes/` 目录下创建笔记文件。

## 注意事项

- 定期整理和更新资料
- 标记已失效的链接
- 保持结构清晰
