# 作业模板库

此目录存储常用的作业模板，可以快速应用到新任务中。

## 支持的模板类型

### 编程模板
- `report_template.py` - Python代码文档模板
- `class_template.py` - 类定义模板
- `algorithm_template.py` - 算法实现模板

### 写作模板
- `report_template.md` - 实验报告模板
- `essay_template.md` - 论文模板
- `proposal_template.md` - 提案模板

## 使用方法

1. 选择合适的模板
2. 复制到你的作业目录
3. 根据需求修改

## 自定义模板

如果你有常用的格式或结构，可以添加新的模板文件：
1. 在对应分类目录下创建新文件
2. 遵循命名规范
3. 在README中简要说明
