# 作业助手 (Homework Assistant)

一个智能化的作业辅助工具，帮助你高效完成课程作业。

## 功能特性

- **任务分析** - 智能解析作业要求，提取关键信息
- **自动规划** - 自动拆解任务，制定执行计划
- **编程辅助** - 代码生成、测试、调试、文档编写
- **写作辅助** - 大纲生成、内容撰写、结构优化
- **记忆功能** - 记住课程知识，对话历史，过往经验
- **自动化执行** - 根据任务类型自动选择最优执行路径

## 使用方法

1. **开始新作业** - 告诉我你的作业要求
2. **查看分析结果** - 我会解析任务并展示拆解计划
3. **确认执行** - 确认后自动执行直到完成
4. **获取结果** - 获得完整的作业解决方案

## 支持的作业类型

- 编程类作业（代码实现、算法题目、系统设计）
- 写作类作业（报告、论文、文档、文献综述）

## 注意事项

- 提供尽可能详细的作业要求，包括评分标准
- 如有参考资料，请一并提供
- 可以随时中断执行并调整计划
